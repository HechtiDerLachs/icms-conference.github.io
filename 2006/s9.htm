Session 9

Title: Methods and Software for Computing Mathematical Functions
Organizers: Amparo Gil (amparo.gil@unican.es)
                      Javier Segura (javier.segura@unican.es)

Description:

Software for computing mathematical functions is included in any imaginable mathematical software package. The development of new algorithms and the improvement of existing software, is a continuously ongoing effort in the field of computational mathematics.
 
This session will be devoted to present recent progress in the development of methods and software for computing mathematical functions, from elementary to several variable functions.
