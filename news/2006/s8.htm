Session 8

Title: Software for Optimization and Geometric Computation
Organizers: Komei Fukuda (fukuda@ifor.math.ethz.ch)
                      Michael Joswig (joswig@mathematik.tu-darmstadt.de)

Description:

In this session, we focus on mathematical software (standalone codes, libraries, languages, computing environments, add-ons for larger systems, etc.) for optimization and geometric computation.  We are also interested in applications of such software packages to other areas such as algebra, statics, control theory, game theory, logic,
computational biology.  Specific topics include:

1. software for linear, combinatorial and non-linear optimization.

2. software for polyhedral computation (convex hull, triangulations, Voronoi diagrams, etc).

3. integration of optimization/geometric software into larger software systems.

4. applications of optimization/geometric software.

5. software libraries for parallelizing optimization, searching and enumeration algorithms.

6. visualization, user interface, modeling languages for optimization and geometric computing.

All forms of software are welcome, but open source software is preferred.
