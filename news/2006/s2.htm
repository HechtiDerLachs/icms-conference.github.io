Session 2

Title: Interfacing Computer Algebra and Mathematical Visualization
Organizer: Konrad Polthier (polthier@mi.fu-berlin.de)

Description:

The session will discuss the interface between computer algebra systems and visualization tools. Traditionally, computer algebra systems focus on exact algebraic calculations while visualization tools fully rely on floating point arithmetic. The session will introduce different ongoing developments in mathematical visualization to find a common language and interchange formats. The talks will present innovative solutions for a set of diverse problems from differential geometry, knot theory, calculus of variation to dynamic geometry.
